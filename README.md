### 必修课4：新媒体故事结构

讲故事越来越成为商业社会的必备技能，达到投资，营销，创业，小到日常的沟通，工作的汇报，我们都在故事中传递我们的经验和认知。但是在讲故事开始越来越被提及，甚至到被滥用的时候，所谓「讲故事」到底在说的是什么？讲故事是最强大的说服工具，故事通过情感的引导，最后再召唤我们的行动。故事的强大之处在于，你无需做任何的辩证。讲故事是一种上帝的视角，你站在宇宙中心去传达你的价值观。 

一个优秀的故事，需要从优秀的人物（People）开始，从目的（Purpose）为终点，通过情节（Plot）点来补补推进。故事结构适用于非常多的场景，譬如节目、访谈、演讲，甚至是对话，不胜枚举。在这一门课程中，我们将学习优秀的人物要具备三大特征，如何通过关键字构建故事目的，最后通过核心冲突和6个基本情节点来构建情节。

### 必修课7：新媒体内容营销

对于时间越来越少、可接收信息越来越多、注意力被不断分散的消费者来说，传统的营销方式正在迅速地失效。内容营销正在成为了新媒体营销的新宠：越来越多的人开始生产免费的内容，为什么有的人成功？有的人失败？关于内容营销，我们存在着太多的误解，它既不是内容植入，也不是内容规划，更不是社会化营销。

半撇私塾是一所项目制在线新媒体大学！所有课程都是基于世界 TOP 50 顶尖新媒体学院（香港城市大学、香港浸会大学等）的课程体系，并与来自中国领先企业（京东、网易等）的专家联合开发。我们强调“做是最好的学”，通过翻转课堂、「PBL」项目制学习、1V1辅导、作品集产出等教学服务，我们致力于通过短期、高强度的项目训练，帮助零基础学员成为行业抢手的全栈新媒体人才！

### 训练营适合零基础学员吗？

训练营就是为没有或只有较少营销经验的学员设计的！无论你是零基础、想要转行还是刚刚完成转行，相关的营销经验在1年以下，如果你愿意每周花5个小时以上的时候进行学习，你都非常适合半撇私塾的训练营课程！

### 学习训练营需要投入多少时间？

具体来说分为两个情况：

1. 如果你是兼职学习，每周可以有5个小时以上的学习时间，并选择购买**「全栈新媒体骇客训练营」标准版的课程，**那么通常需要6个月左右的时间来完成所有的课程；
2. 如果你是全职学习，每天拥有5个小时以上的学习时间，并选择购买**「全栈新媒体骇客训练营」尊享版**的课程，那么通常来说**2个月**可以学习完所有的课程并毕业；

### 学完训练营可以达到什么程度？

从知识和技能上，全栈新媒体骇客训练营可以通过6个月高强度的项目实战，帮助你系统掌握应用传播、故事、产品、技术、短视频与增长黑客等职业所需技能。

从求职资历上，全栈新媒体骇客训练营可以帮助你产出从创意、产品到种子用户的完整项目作品集，它可以为你的求职增加许多谈判的筹码。

如果你成功地从训练营毕业，您将能够胜任中国一线城市月薪8K+的新媒体运营岗位。

### 训练营是什么样的授课方式呢？

半撇私塾的课程采用「线上录播视频」+「在线互动测试」+「在线结业考试」+「实战项目作业」的形式。我们对课程体系和内容进行反复的迭代和优化，确保你学习到的都是最优质的内容！多年的教学实践也证明，录播课程的学习效果要比一次性的线下培训更好：当你在学习中遇到困难，可以暂停播放，反复学习直到学会为止。

更具体的来说，在整个训练营的学习期间，你都需要：

1. 学习由“课程导师”制作的视频课程，掌握核心知识；
2. 参与在线课程测试，巩固核心知识点；
3. 完成在线结业考试，检验知识掌握程度；
4. 完成实战项目作业，提交“项目导师”批改，并根据批改调整完善项目作品；

在整个学习的过程中，你可以享受无限次数的教学答疑和不定期的在线直播答疑服务。如果你顺利地完成所有的课程项目，你可以申请进入职业辅导阶段，老师会根据你的求职目标为你修改简历、优化求职作品集、提升求职背景、修改面试话术、推荐合作企业的岗位等。

### 我该选择训练营哪个版本，有什么区别？

你可以根据自身的目标和学习时间来选择课程版本：

1. 如果你只是对新媒体感兴趣，想要了解系统了解媒体营销，那么「基础版」会更适合你。
2. 如果对新媒体有一定的认识，但是缺乏实战经验，并且在未来的时间想要系统学习新媒体，但是对求职不是非常迫切（至少未来3个月内不着急找工作），那么「标准版」会更适合你（在半撇私塾的学员构成中，90%人都选择标准版）。
3. 如果你每天有5小时以上的学习时间，并且具备强烈的学习欲、较强的学习能力，并期望能在最短的时间内完成学习，找到新媒体相关的工作，那么「尊享版」课程会更为合适你。

不同版本的对比和区别可以查看 [价格表](https://www.bpteach.com/pricing/)。

### 训练营是随到随学吗？

训练营是随到随学的，你可以根据自己的时间安排进行课程的学习。当然我们会根据历史学员的学习情况，为你制定每一门课程的学习时间表，以帮助你更好地跟踪学习进度。另外，我们还有邮件、社群、短信等多维度的提醒机制，督促你完成课程的学习。

### 训练营课程有效期多长呢？

对于许多购买在线课程的学员来说，常常因为“永久有效”，觉得什么时候都可以看，付费购买后就没有再学习了。因此为了保证学习的效率和质量，我们对训练营的课程都设定了具体的有效期。在课程开始前，我们都会帮助你制定每一门课程的学习时间安排，以帮助你更好地跟踪学习进度，当然有时候落后一点点进度也没有关系，你可以在有效期内自行安排任意时间赶上进度！

担心不能顺利毕业？在你正式进入课程学习后，我们会有专业的助教、导师为你提供课程答疑、项目审阅、闯关学习小组等服务，绝大多数同学都能在有效期内完成课程。当然，一方面我们会尽可能帮助你，另外一方面也需要你个人的努力。如果你在有效期内顺利毕业，我们会为你**免费赠送额外180天**视频观看权限，方便毕业学员进行复习。

如果有效期过后你还没有学完课程，你便无法继续观看课程视频，你可以选择 [按月续费](http://learn.bpteach.com/course/291)（只要在有效期结束7天内续费，你都可以可以享受[「30%学费返还计划」](https://www.bpteach.com/knowledge-base/1011371/) ）

### 学习训练营的先修要求是什么？

这是一门面向零基础学员的课程，你只需要具备如下条件：

- 具备充分的学习自主性，热爱动手并每周投入时间学习此课程，能够在截止日期前完成作业和项目要求后才能获得毕业证书。
- 拥有基本的计算机操作技能，例如使用浏览器访问互联网，使用搜索引擎，使用浏览器翻译功能，下载和安装程序，文件重命名等。
- 能够参与视频会话（例如使用微信视频聊天或Skype），与助教交流沟通（在1V1辅导时候需要）。

课程中会涉及部分软件和工具的操作，我们都会推荐不同系统和平台的产品，所以学员无需为了配合课程内容购买产品。
