# 「全栈新媒体运营微专业」学习计划表

这是一段充满挑战的学习旅程，每一周，你需要学习新知识，掌握新技能，并且需要适当地在 Github 社区、学习社区、微信群中跟小伙伴们分享你的学习收获。
